• Foreman fusor_v21

Satellite API v1 is deprecated. Please use v2. If you still need to use v1, you may do so by either passing ‘version=1’ in the Accept Header or using api/v1/ in the URL.

Resources

Customer Portal Proxy

Resource Description
post /fusor/api/customer_portal/login Log in to customer portal (creates session)
post /fusor/api/customer_portal/logout Log out of customer portal (deletes session)
get /fusor/api/customer_portal/is_authenticated Verifies an active authenticated session to the customer portal.
get /fusor/api/customer_portal/users/:login/owners Get a list of subscription owners from RHN customer portal. See customer portal API documentation for a full list of parameters.
get /fusor/api/customer_portal/pools Get a list of subscription pools from RHN customer portal. See customer portal API documentation for a full list of parameters.
get /fusor/api/customer_portal/owners/:id/consumers Get a list of subscription consumers from RHN customer portal. See customer portal API documentation for a full list of parameters.
get /fusor/api/customer_portal/consumers/:id Get a subscription consumer from RHN customer portal. See customer portal API documentation for a full list of parameters.
post /fusor/api/customer_portal/consumers Create a new subscription consumer from RHN customer portal. See customer portal API documentation for a full list of parameters.
get /fusor/api/customer_portal/consumers/:id/entitlements Get a list of subscription entitlements from RHN customer portal. See customer portal API documentation for a full list of parameters.
post /fusor/api/customer_portal/consumers/:id/entitlements Create new subscription entitlement from RHN customer portal. See customer portal API documentation for a full list of parameters.
delete /fusor/api/customer_portal/consumers/:id/entitlements Delete all entitlements for a subscription consumer from RHN customer portal. See customer portal API documentation for a full list of parameters.
